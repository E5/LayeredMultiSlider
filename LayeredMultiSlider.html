LayeredMultiSlider a combined SCView


*new(containerView, width, height, numSliders, numLayers, colours, layerNames);


containerView: an SCWindow or SCCompositeView

width width of the LayeredMultiSlider (total = width + 8)

height height of the LayeredMultiSlider (total = height + 32)

numSliders number of sliders in each SCMultiSlider

numLayers number of layers to use

colours an (optional) array of colours for each layer

layerNames an (optional) array of names for each layer


*value(layer, slider) get value of single slider in layer

*valueArray(layer) get value of all sliders in layer

*value_(layer, slider) set value of single slider in layer

*valueArray_(layer) set value of all sliders in layer


// Example


(

// How to add a LayeredMultiSlider to a window


w= SCWindow("LayeredMultiSlider test", Rect(128, 227, 500, 200)).front;


w.view.decorator= FlowLayout(w.view.bounds);


n= LayeredMultiSlider(

w, // containerView

(w.bounds.width-16), // width

60, // height

16, // numSliders

4 // numLayers, colours, layerNames

);

)


// run this line to set the value of a single slider in a layer:

n.value_(1, 4, 0.666);


// run this line to get the value of a single slider in a layer:

n.value(1, 4);


// run this line to set the value of all the sliders in a layer:

n.valueArray_(1, Array.fill(16,{1.0.rand}));


// run this line to get the value of all the sliders in a layer:

n.valueArray(1);
